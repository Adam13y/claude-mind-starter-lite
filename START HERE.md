# START HERE

Five minutes, and then you're done setting up forever.

## 1. Open this folder as an Obsidian vault

Obsidian → *Open folder as vault* → pick this folder. (Obsidian is optional, honestly — these are plain markdown files and Claude doesn't care what you read them in. But the `[[wikilinks]]` only become clickable in Obsidian.)

## 2. Give Claude access to the folder

Claude Code, or Claude Desktop with a connected folder — anything that can read and write files in this directory.

## 3. Say this

> Read CLAUDE.md and tell me what I'm working on.

Claude reads the protocol, reads your task files, and tells you where you stand. On a fresh vault it'll tell you it's empty — that's correct, and it's the last time you'll have to explain anything twice.

## 4. Just work

Talk to Claude normally. When you commit to something, it lands in `Tasks/TASKS.md` without you asking. When you're waiting on someone, it lands in `Open Loops`. That's the whole point: **you shouldn't have to remember to make Claude remember.**

## 5. End the session with three words

> update your mind

Claude closes out what's finished, archives it, updates what changed, and writes down what it learned. Next session starts from there instead of from nothing.
